# Training

This directory contains educational resources and training materials for the IAMX platform.

## CloverDX Exercises

The `cloverdx-exercises/` directory contains the CloverDX training exercises that demonstrate:

- Basic ETL operations
- Data transformation patterns
- API integration
- Error handling
- Database connectivity
- File processing

### Getting Started with Exercises

1. Import the `cloverdx-exercises` workspace into CloverDX Designer
2. Follow the exercise instructions in each graph
3. Experiment with the provided data samples
4. Apply concepts to your clinic projects

## Integration with IAMX

These exercises complement the IAMX utilities by providing hands-on practice with:

- Data flow design patterns
- Transformation logic implementation
- Error handling strategies
- Performance optimization techniques
